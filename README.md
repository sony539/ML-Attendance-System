# ML Attendance System

## Project: Building Attendance System using ML
## Internship: ML using Python - APSSDC

### Description:
Attendance system for 980 students who attended 40 sessions.
Students with average >= 96 mins (80% of 120) are Eligible.

### Steps:
- Data Collection
- EDA
- Preprocessing
- Visualization
- Logistic Regression (100% Accuracy)

### Results:
- Eligible     : 739 students
- Not Eligible : 241 students
